\# AI Traffic Monitoring System



Computer vision project that detects, tracks and counts vehicles in a video using YOLOv8.



\## Features



\- Vehicle detection

\- Object tracking

\- Line crossing detection

\- Vehicle counting

\- Real-time visualization



\## Technologies



\- Python

\- YOLOv8

\- OpenCV

\- PyTorch



\## Demo



![Demo](demo.gif)



\## Installation



Clone the repository:



git clone https://github.com/MariaPag19/traffic-monitoring-ai.git



Install dependencies:



pip install -r requirements.txt



Run the program:



python traffic\_analyzer.py



\## How it works



1\. YOLO detects vehicles in each frame

2\. Each vehicle receives a unique ID

3\. When the vehicle crosses the line it is counted
